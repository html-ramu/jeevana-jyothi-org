# Jivana Jyoti School for the Blind 🌟

Welcome to the official repository for the **Jivana Jyoti School for the Blind** website. This project is a responsive, accessible, and beautifully designed web interface aimed at showcasing the mission, programs, and achievements of the Jivana Jyoti institute, which empowers visually impaired students in Andhra Pradesh and Telangana.

## 🚀 Features

- **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.
- **Image Carousel:** A dynamic, auto-playing slideshow highlighting student activities (Sports, Technology, Music, Yoga, etc.).
- **Bilingual Content:** Seamlessly integrates English and Telugu text using native fonts (`Sree Krushnadevaraya`, `Mallanna`).
- **Accessibility Focused:** Designed with clean UI, readable typography, and semantic HTML to ensure a good experience.
- **Interactive Sections:** Includes Programs, Milestones (Counters), Alumni Testimonials, and a Contact form.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure and accessible elements.
- **CSS3:** Custom properties (variables), Flexbox, CSS Grid, and responsive media queries.
- **Vanilla JavaScript:** For carousel functionality, mobile hamburger menu, scroll-to-top button, and dynamic stat counters.
