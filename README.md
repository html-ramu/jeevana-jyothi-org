# Jeevana Jyothi Inclusive School 🪔

Welcome to the official repository for the **Jeevana Jyothi Inclusive School, Nandyal** website. Organized by the **Louis Braille Blind Welfare Association**, this project is a fully responsive, accessible, and beautifully designed web interface aimed at showcasing the mission, programs, and achievements of our institute, which empowers visually impaired and underprivileged students in Andhra Pradesh and Telangana.

## 🚀 Features

- **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.
- **Image Carousel:** A dynamic, auto-playing slideshow highlighting student activities, events, and milestones.
- **Trilingual Content:** Seamlessly integrates English, Hindi, and Telugu text using native fonts (`Hind`, `Mallanna`).
- **Accessibility Focused:** Designed with clean UI, readable typography, and semantic HTML to ensure a great experience.
- **Comprehensive Sections:** Includes Programs, Milestones (Counters), Alumni Testimonials, Founders' Message, AIMS & Objectives, Brochure, Image Gallery link, and a Contact form.
- **App Ready:** Code is structured to be easily converted into an Offline WebView Android App.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure and accessible elements.
- **CSS3:** Custom properties (variables), Flexbox, CSS Grid, and responsive media queries.
- **Vanilla JavaScript:** For carousel functionality, mobile hamburger menu, scroll-to-top button, form handling, and dynamic stat counters.
